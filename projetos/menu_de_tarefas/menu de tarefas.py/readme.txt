# Menu de Tarefas

Projeto de gerenciamento simples de tarefas em Python, com estrutura de menu interativo e gravação em arquivo `.txt`.

## Funcionalidades
- Adicionar tarefas
- Listar tarefas
- Remover tarefas
- Salvar e carregar tarefas em arquivo

## Como usar
Execute o `main.py` com Python 3.11+
